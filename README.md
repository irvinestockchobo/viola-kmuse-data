# viola-kmuse-data

K-Muse curated assets for [kmusestyle.com](https://kmusestyle.com).

## Image CDN Usage

Use jsDelivr for fast CDN delivery:

```
https://cdn.jsdelivr.net/gh/irvinestockchobo/viola-kmuse-data@main/images/fashion/filename.jpg
```

## Folder Structure

```
images/
  fashion/     — K-fashion trend images
  travel/      — Korean travel & destinations
  food/        — K-food & restaurant imagery
  beauty/      — K-beauty products & looks
  lifestyle/   — General K-lifestyle content
data/          — Curated data files (JSON, CSV)
```

## Naming Convention

`[category]-[keyword]-[YYYYMMDD].[ext]`

Example: `fashion-dustbag-trend-20260416.jpg`
